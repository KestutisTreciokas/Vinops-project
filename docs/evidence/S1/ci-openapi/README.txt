OpenAPI CI Evidence
===================

Назначение каталога: хранение локальных копий отчётов и bundled-спеки, полученных CI или локальным прогоном.
Стандартные имена:
- oas-validate.txt
- oas-lint.txt
- openapi.bundled.yaml

Полные артефакты каждого прогона доступны также в GitHub Actions (Artifacts) у job "Contract tests (OpenAPI v1)".
